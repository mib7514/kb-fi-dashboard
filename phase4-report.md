## 1. 계산 (v1 3Y 단일값 → 구간별 일반화)

```
v2 구간별:  E[Δy_구간] = Σi P(금리 i) · sceneCurve[dir i][구간]        (expectedDyByTenor)
혼합:       E[Δy_구간]_mix = w·E[Δy]평행(3Y 스칼라) + (1−w)·E[Δy]v2[구간]   (mixEDy)
커브이동:   −D′(구간) × E[Δy_구간]_mix                                   (decompose, 배열 eDy)
1층 기본값: default[dir][구간] = Σj P(스프레드 j) · medianCurves[dir|j].deltaBp[구간]  (conditionalDefaultCurves)
```
- w=100% → 혼합 = 평행(v1)과 동일. w=0% → 순수 v2. (단위테스트로 등가성 고정 — 체크리스트 ③)
- `decompose(curveY, eDy)`는 eDy가 스칼라(평행)든 배열(구간별)이든 처리 → v1 경로 무손상.
