## 1. 계산 정의 (구간별, 1개월 보유, bp)

```
기대수익(bp) = 캐리 + 롤다운 + 커브이동
  캐리     = y0 × 100 / 12
  롤다운   = −D′ × (y_현재커브(만기−1/12) − y0) × 100      … 착지커브 = 현재커브
  커브이동 = −D′ × E[Δy]                                    … 평행: E[Δy] 전 구간 동일
  D′ = 만기 − 1/12        (수정듀레이션 ≈ 잔존만기, 쿠폰 미반영 v1, §6 보류)
  E[Δy] = Σi Σj P(금리i)·P(스프레드j)·medianCurves[i|j].deltaBp[3Y]   (RG-1 확률 실시간 참조)
```

- 커브 보간: 8점 격자 [0.25,0.5,1,1.5,2,2.5,3,5]y 선형, **3M 미만은 3M 값 평탄 외삽**.
- 확률 미입력(축 합 0) 또는 medianCurves 미로드 → E[Δy]=null → 커브이동 0 + 안내(에러 없음).

## 2. 검산 — 2Y 구간 손계산 (코드 결과와 일치)

**입력:** 예시 커브(%) 3M=3.30 / 6M=3.35 / 1Y=3.40 / 1.5Y=3.45 / 2Y=3.50 / 2.5Y=3.56 / 3Y=3.60 / 5Y=3.75.
RG-1 기본 확률 금리[33,34,33] 스프레드[33,34,33]. medianCurves = 커밋된 `data/rg-calib.js` 3Y 열.

**Step 1 — E[Δy] (평행, 3Y 중위 Δ 확률가중)**

3Y 중위 Δbp(셀별): 하락[−11.5, −11.5, −13.3] · 보합[0.5, 0.2, −0.9] · 상승[13.6, 14.0, 17.0]
(열 = 축소/보합/확대). 가중치 wᵢⱼ = (금리ᵢ/100)(스프레드ⱼ/100).

- 하락 행 Σⱼ = 0.33·(−11.5) + 0.34·(−11.5) + 0.33·(−13.3) = −12.094 ; ×0.33 = **−3.99102**
- 보합 행 Σⱼ = 0.33·0.5 + 0.34·0.2 + 0.33·(−0.9) = −0.064 ; ×0.34 = **−0.02176**
- 상승 행 Σⱼ = 0.33·13.6 + 0.34·14.0 + 0.33·17.0 = 14.858 ; ×0.33 = **+4.90314**
- **E[Δy] = −3.99102 − 0.02176 + 4.90314 = 0.89036 bp**  (코드: 0.8903600)

**Step 2 — 2Y 3성분** (T=2, D′ = 2 − 1/12 = 1.916667)

- 캐리 = 3.50 × 100 / 12 = **29.16667 bp**
- 착지: 만기−1/12 = 1.916667 → 1.5Y(3.45)~2Y(3.50) 보간, t=(1.916667−1.5)/0.5=0.833333
  → y_land = 3.45 + 0.833333×0.05 = **3.491667**
- 롤다운 = −1.916667 × (3.491667 − 3.50) × 100 = −1.916667 × (−0.833333) = **+1.59722 bp**
- 커브이동 = −1.916667 × 0.89036 = **−1.70652 bp**
- **기대수익 = 29.16667 + 1.59722 − 1.70652 = 29.05737 bp** (표시 29.1)

**코드 산출값 (js/rg-rolldown.js):** carry 29.166667 · rolldown 1.597222 · curveMove −1.706523 · total 29.057366
→ **손계산과 일치** (허용오차 ±0.001 단위테스트 통과).

**순위(예시 커브):** 5Y 29.9 · 2.5Y 29.9 · 3Y 29.3 · 2Y 29.1 · 1.5Y 28.7 · 1Y 28.3 · 6M 28.2 · 3M 27.4
→ 최고 기대수익 **5Y** (막대·순위표 하이라이트).
