전향적 레짐 판단

금리·스프레드 방향 확률 → 9셀 결합확률 히트맵으로 향후 1개월 레짐을 전향적으로 판단합니다. 근거(why)는 기록하지 않고 무엇을(what) 예상하는가만 — 주간 확정 후 1개월 뒤 실현값과 대조(채점은 RG-4).

❓ 이 화면 읽는 법 — 9셀 히트맵이 뭔가요?

무엇을 입력하나

향후 1개월 금리 방향 확률 3개(하락/보합/상승, 국고 3Y 기준)와 스프레드 방향 확률 3개(축소/보합/확대, 회사채 AA- 3Y − 국고 3Y 기준). 각 축 합계는 100%가 되어야 확정할 수 있습니다. 슬라이더를 움직여도 다른 칸은 자동으로 바뀌지 않습니다(수동 입력).

어떻게 판단으로 바뀌나

9셀 결합확률 = P(금리 i) × P(스프레드 j). 확률이 가장 높은 셀이 최빈 레짐이며, 하단 플레이북에 국면별 정석 대응이 상시 표시됩니다.

보합 기준

"보합"은 1개월 변화의 절댓값이 캘리브레이션 보합밴드보다 작은 경우입니다(입력 가이드 참조). 입력 시와 채점 시 동일 밴드를 씁니다.

주의

두 축은 독립으로 곱합니다(v1 단순화). 실제로 금리와 스프레드는 상관될 수 있으며, 조건부 확률은 후속 버전 검토 항목입니다.

금리 방향 · 국고 3Y
합계 —

정규화 버튼은 입력값 자체를 합계 100%로 재기입합니다(비례 배분). 정규화 전에는 확정 불가.
9셀 결합확률 히트맵 — P(금리)×P(스프레드), 색 농도=확률, 최빈 셀 강조
9셀 결합확률은 두 축의 독립 곱입니다(v1 명시적 한계). 셀 색은 국면 성격(우호/중립/방어), 농도는 확률입니다.
플레이북 전문 — 9국면 정석 대응 (상시 표시)
이론·교과서적 정석입니다. 특정 배분 테이블·레버리지 가이드는 포함하지 않습니다.
롤다운 탐색기 — RG-2 · 1개월 보유 구간별 기대수익(bp) = 캐리 + 롤다운 + 커브이동
국고 커브 입력 · 구간별 민평 수익률(%) 세션 전용 · 저장되지 않음
시나리오별 커브이동 (v2) — 앵커 모드: 3M·3Y 착지금리 2점 → 9레짐 조건부 모양으로 나머지 자동 산출 · 단위 Δbp
100% w=100% 평행(v1) · w=0% 내전망(v2) · 그 사이 혼합(shrinkage)
구간별 기대수익 — 혼합 기준 · 캐리 / 롤다운 / 커브이동 3성분 (좌=음수, 우=양수)
캐리 롤다운 커브이동 · 좌측(빗금)=음수 성분
v1 / v2 / 혼합 비교 — 구간별 기대수익(bp)
순위 — 혼합 기대수익 내림차순
롤다운 = 착지커브를 현재커브로 가정. 커브이동 = w×평행(v1) + (1−w)×내전망(v2), 구간별 E[Δy]에 D′(≈잔존만기) 곱. 앵커 모드(기본): 시나리오별 3M·3Y 착지금리 2점만 입력하면 9레짐 조건부 모양에 affine 보정(a·shape+b)해 나머지 6구간을 자동 산출합니다(앵커 2점은 정확 일치, 중간은 기본커브 곡률 보존). RG-1 스프레드 확률을 바꾸면 모양(shape)이 바뀌어 파생 커브가 갱신되지만 앵커 2점은 고정됩니다. 상세 모드로 전환하면 24칸을 직접 편집할 수 있습니다(기존 방식). 듀레이션은 만기 근사(쿠폰 미반영). 국고 커브·착지금리(레벨 %)는 저장되지 않으며 새로고침 시 소실됩니다(앵커 Δbp만 작업본에 저장 — §0.3).
섹터 스프레드 보드 — RG-3 · 6섹터 방향 확률(축소/보합/확대) → 기대 Δs 순위
각 섹터 방향 확률을 입력하면 기대 스프레드 변화 E[Δs] = δ×(P확대 − P축소)로 매력 순위를 냅니다(δ = 섹터 보합밴드). 축소 기대(음수)일수록 매력 상위. ⇄ 표시된 국고채·회사채 행은 RG-1 축과 동일 입력(국고채=금리 축, 회사채=스프레드 축)이라 어느 쪽에서 고쳐도 즉시 양쪽에 반영됩니다. 공사·은행·카드·여전채는 기본적으로 RG-1 스프레드 축을 상속(RG-1 따름)하여 대표 스프레드 뷰에서 출발하고, 어느 칸이든 수정하면 그 섹터만 개별 조정으로 전환·동결됩니다("RG-1 값으로 리셋"으로 복귀). follow 섹터는 스프레드 축이 정규화돼 있으면 별도 정규화가 필요 없습니다.
국고채 밴드(±4.6bp)는 금리축(국고 3Y 수익률 변화)과 동일 계열입니다 — 국고 행은 RG-1 금리 축과 같은 입력이며 하락=축소 열, 상승=확대 열로 매핑됩니다("축소/확대"는 수익률 방향). 확률은 각 섹터 합계 100%여야 확정됩니다.
섹터×구간 매력도 매트릭스 — 캐리 + 롤다운 + 확률가중 금리·스프레드 이동 = 섹터×구간 총 기대수익(bp)
각 섹터 현재 커브 = 국고 커브 + 섹터 스프레드(만기 평탄 가정). 총 기대수익 = 캐리(섹터 수익률) + 롤다운(국고 모양) − D′×(E[Δy_구간] + E[Δs_섹터]). E[Δy]는 위 RG-2의 혼합 w 설정을 그대로 씁니다. 국고채 행은 스프레드 0·E[Δs]=0(순수 금리)이라 위 RG-2 순위와 일치합니다. 국고 커브 8구간을 입력해야 매트릭스가 계산됩니다.
섹터 현재 스프레드 (bp · 국고=0 고정, 수정 가능)
① 현재 금리판 — 섹터×구간 현재 수익률(%) · 세션 표시만(저장 안 함)
② 1개월 후 착지판 — 확률가중 착지 수익률(%) · 세션 표시만
③ 매력도 비교 — 섹터×구간 총 기대수익(bp) 스프레드 커브 평탄 가정
최고 셀(테두리 강조)이 판단 시점 선택 셀입니다. 섹터별 최고 구간은 굵게 표시됩니다. 스프레드 레벨(bp)은 커밋된 공개 파생 데이터라 저장 가능하나, 수익률 레벨(%)은 세션 표시만이며 저장·스니펫에 담지 않습니다(§0.3).
