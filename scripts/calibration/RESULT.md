# Taylor 압력 캘리브레이션 결과 (RESULT.md)

측정 레이어 · 재현 검증용 리포트. 파라미터는 이미 동결됨(의사결정 아님).
재현 스크립트: `node scripts/calibration/calibrate.mjs` (레퍼런스 `reference.csv`).

## 1. 방법

- **레퍼런스**: 사용자 엑셀 모델 산출 압력 `reference.csv` (52포인트, 2015-03 ~ 2025-12 — 2015Q1~2024Q4 분기말 + 2025 월별).
- **모델**: `i* = r* + π + α·(π − 2.0) + β·ygap`, `압력 = i* − 기준금리`
  - `π` = 근원 CPI YoY(%), `ygap` = (ln 실질GDP − **one-sided** HP추세(λ=1600))×100 [분기→월 forward-fill, 미발표분 직전분기 hold]
  - 기준금리 = 해당 월말(월 마지막 관측) 정책금리
- **데이터(ECOS, 라이브 검증)**: 근원 CPI `901Y010`, 실질SA GDP `200Y104/1400`, 기준금리 `722Y001/0101000`. HP 워밍업 GDP 2000Q1~, CPI YoY용 2013-01~.
- **지표 관례**(노트북과 동일):
  - **방향 일치율** — 스펙의 "분기 변화 방향". 분기말 포인트만, 레퍼런스 무변화 분기쌍은 분모 제외(2017-09→12, 2019-06→09) → 분모 41.
  - **상관·RMSE** — 전 겹침구간("분기말/월말", 52포인트) 대상.
- **그리드**: r* ∈ [−1.0, 1.5]/0.1, α ∈ [0.25, 1.0]/0.05, β ∈ [0.25, 1.0]/0.05.
- **채택 게이트**: (1) 방향 ≥ 70% → (2) 상관 ≥ 0.85 → (3) 통과 조합 중 RMSE 최소.

## 3. 채택안 및 재현 판정

**동결 파라미터: r* = 1.20, α = 0.25, β = 0.45** (π\* = 2.0, λ = 1600).

- 그리드 최적 = 동결안과 **정확히 일치**.
- 재현값 **방향 75.6% / 상관 0.924 / RMSE 0.473** — 노트북(Python) 결과 **75.6% / 0.924 / 0.473 와 완전 일치**(Δ 방향 0.0%p, Δ상관 0.000, ΔRMSE 0.000). → 파라미터 동결 유지.

## 4. 명시 사항 (v1 한계 · v2 재점검)

1. **headline CPI(총지수) 기각** — 직전 캘리브레이션에서 방향 **82.9%**로 높았으나 상관 **0.816**(< 0.85)으로 기각. 본 재현에서도 상관 0.816으로 동일 재현되어 상관 게이트에서 탈락(재현 방향은 78.0%로, 방향 지표는 관례 차로 소폭 달라지나 기각 사유인 상관은 정확히 일치).
2. **OECD 방식 core(식료품·에너지 제외, DB) 기각** — 그리드 최적의 **r\*가 그리드 상단 경계(1.50)에 붙음** → 경계 해(under-identified)로 신뢰 불가하여 기각.
3. **채택안 α=0.25는 그리드 하단 경계값** — 최적이 경계에 붙어 있어, v2에서 **α 그리드 하향 확장(예: 0.0~)** 재점검 대상. (β=0.45는 내부 해로 안정.)
